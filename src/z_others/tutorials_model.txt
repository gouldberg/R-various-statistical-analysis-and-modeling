# ------------------------------------------------------------------------------
# 計量経済学におけるモデル
# ------------------------------------------------------------------------------

- 多項式ラグ・モデル（アーモン・ラグ・モデル）
  - 分布ラグの係数を、低次（通常は二次から四次程度）の多項式で近似

  - 係数に研究者の恣意性を持たせずに推定

　　- パラメータ数の大幅な節約ができ、自由度の不足や多重共線性が多少なりとも改善される

　　- 長いラグが必要とされるケースで検討：　輸入・消費行動のように習慣形成がある、石油や一次産品など長期契約に基づく取引が影響するなど



- ラグ付き内生変数を含むモデル

  - 幾何級数型の分布ラグ・モデルは、ラグ付き内生変数を含むモデルとして表せる
　　　　※適応型期待形成に基づく期待変数を説明変数とするモデルは、幾何級数型の分布ラグ・モデルを導く

  - 幾何級数型モデルを、ラグ付き内生変数を含むモデルとして表すと、撹乱項が一期前の撹乱項を含む移動平均型になるため、最小二乗法推定量は不偏推定量でも一致推定量でもない

  - 実証研究では、部分調整モデル（※）を想定し、撹乱項が系列相関を持たないと仮定することが多い
　　部分調整モデルでは、最小二乗法推定量は、不偏推定量ではないが一致推定量となる　（説明変数と撹乱項の相関がゼロの場合は最小二乗法推定量は不偏性は持たないが一致性は持つ）
　　　　※最適な量と一期前の差の一部分しか今期には調整できない、瞬時の調整ができず目標に向かい徐々に調整が行われるとする　⇒　　部分調整モデルでは、撹乱項が無限の過去を含むことになる

  - 撹乱項に系列相関を持つモデル（下記のAR(1)など）は、部分調整型のラグ付き内生変数を含むモデルとして表現できる

　　- 多項式ラグ・モデルにラグ付き内生変数を加えることで、より複雑なラグ構造を推定することも可能



- ラグ付き内生変数を含むモデルの撹乱項に系列相関がある場合

　 - ラグ付き内生変数を含むモデルでの撹乱項の系列相関は、ダービンのh統計量 あるいは ダービンの代替的方法　で検定
　    - ダービンのh統計量は、分母が負値になると計算不可となってしまう
　    - 代替的方法は、撹乱項が高次のラグ構造を持つモデルの検定に拡張できる

　 - このモデルの撹乱項に系列相関があるということは、内生変数と撹乱項が、一期前の撹乱項を含むことになり、説明変数と撹乱項に相関があることになる

  - ラグ付き内生変数を含めても系列相関が残る場合は、操作変数法で推定する　（不偏推定量ではないが一致推定量が得られる）
　　あるいは、撹乱項に正規分布を仮定して最尤法により推定する　（ただし非線型推定の手法が必要）

  - 操作変数法での推定方法：
　　最初に操作変数法でモデルを推定し、その残差より系列相関係数（rho）の一致推定量を求めて、
　　コクラン＝オーカット流の繰り返しを利用し収束するまで計算する
　　操作変数法の候補は、内生変数以外のラグ付き変数のみ
　　（内生変数のラグ付き変数はいくらラグをとっても撹乱項との相関が消えないので注意）



- 操作変数法

　 - 説明変数と撹乱項に相関がある場合でも、一致推定量が求められる

　 - 最小二乗推定をした理論値を操作変数として用いる時は、操作変数法はその理論値を用いた最小二乗法となる

  - 操作変数法を用いた場合の、ダービン＝ワトソン検定統計量は、厳密な意味ではもはや残差の系列相関の正しい検定統計量ではないが、近似的には系列相関の情報を与えるものとして使うことができる
　　系列相関が確認された場合は、操作変数法とコクラン＝オーカット流の修正の併用が必要となる



- 撹乱項目に系列相関あるケース

　 - 慣性：　GNP、消費、消費者物価などの経済変数は、持続的または循環的な動きをする傾向にある

　 - 振動と収斂：　消費行動などマイナスの自己相関　(大きな出費のあとは大幅減、反動で大きな出費など）

　 - 想定の誤り：　説明変数を誤って除外した場合、除外された変数に自己相関があれば、想定されたモデルの撹乱項はあたかも系列相関があるような動きを示す

　 - ラグ付き内生変数モデルから内生変数が欠落

　 - 関数形の誤り

　 - 移動平均：　移動平均を用いて季節調整後のデータ



- 説明変数と撹乱項の間に系列相関のあるケース　（最小二乗法推定量は、不偏性も一致性も持たない）

　 - 同時方程式モデル

　 - 説明変数に観測誤差がある場合

　 - 合理的期待仮設に従って形成されている将来についての期待値を説明変数としてもつモデルにおいて、期待値の代わりに将来の実績値を用いた場合

　 - ラグ付き内生変数を含むモデルの撹乱項に系列相関がある場合
　    - 内生変数と撹乱項が一期前の撹乱項を含むことになり、説明変数と撹乱項に相関があることになる



- 多重回帰の撹乱項が一階の自己回帰モデルに従う場合（AR(1)）

  - 経験的にこのモデルで多くの経済データの系列相関を記述できることが知られており、計量モデルの撹乱項に系列相関の可能性を検討する場合、95%がこのAR(1)モデル

  - 経済モデルにおいては、（負よりも）正の系列相関を想定する場合が多い

　　- 撹乱項の系列相関は、ダービン＝ワトソン統計量で検定し、
　　系列相関がなければ最小二乗法推定量を用い、あればコクラン＝オーカットの方法で推定する
　　コクラン＝オーカットの方法による推定量は、不偏推定量ではないが一致推定量

　　- あるいは、系列相関係数そのものをモデルに織り込み、非線型推定法（※）で推定することも可能
　　（※）この場合、ラグ付き内生変数モデルとなるが、係数が元のモデルの係数と系列相関係数の積となりパラメータが非線型となる

  - 実際によく用いられるモデルは、単にラグ付き内生変数を加え、系列相関を除去したモデルで推定を試みることが多い
　　例えば、輸入関数の推定で、被説明変数を「実質輸入等」に対し、説明変数を「実質GNP」「相対価格＝輸入デフレーター÷卸売物価指数」などとしていたところ、
　　説明変数に「前期の実質輸入等」を加えることで、系列相関を除去する



- 定常過程（弱定常性）

　　- 期待値は時点に依存せず一定、自己共分散は時点に依存せず時間差のみに依存する　（自己相関も時点に依存しない）

　　- 注意：　条件付き期待値や条件付き分散が時間を通じて一定であることは要求していない
　　　　　- ARMAモデルやGARCHモデルは定常過程であるが、条件付き期待値や条件付き分散は時変的

　　- 例外：　正規過程は弱定常過程として定義されるが、強定常性を有す

　　- 特徴
　　　　　- トレンドがない
　　　　　- 平均回帰性
　　　　　- 過程の分散が有限

　　- 例：
　　　　　- ホワイトノイズ：　すべての時点で期待値がゼロで、かつ分散が一定、自己相関をもたない　（期待値がゼロのiid系列はホワイトノイズ、正規過程のホワイトノイズはiid系列）
　　　　　- 株式収益率



- 強定常性

　　- 時系列の同時分布が不変
　　（任意の時点、時間差に対して同時分布が同一、　すべての形の依存構造が時点に依存せず時間差のみに依存、　すべての高次モーメントについて時刻に無関係なランダム変動）

　　- 例：
　　　　　- iid系列：　各時点のデータが互いに独立でかつ同一の分布に従う系列　（正規過程のホワイトノイズはiid系列）



- エルゴード性

　　- 多くの物理現象は、定常過程であるときエルゴード性をもち、現象の統計的特性は任意の観測サンプルの時間的平均により推定できる




- 移動平均過程（MA過程）

　　- ホワイトノイズの線形和　（MA過程の確率的変動はすべてホワイトノイズで決まる）　⇒　常に定常過程

　　- 期待値と自己相関が同一のMA過程が複数存在するが、反転可能（※）なものは1つのみ
　　（※反転可能：　MA過程がAR(∞)過程に置き換えることが可能、　反転可能のとき撹乱項e(t)は過去のy(t)の関数として表現でき、パラメータの推定や予測を自然な形で行うことができるため反転可能な過程を選択するのが望ましい）

　　- MA特定方程式のすべての解の絶対値が1より大きいとき、MR過程は反転可能



- 自己回帰過程（AR過程）

　　- 過程が自身の過去に回帰された形で表現される

　　- AR特定方程式のすべての解の絶対値が1より大きいとき、AR過程は定常

　　- 定常なAR過程は、MA(∞)過程に置き換えることが可能　（MA係数は指数的に減衰）

　　- AR(p)過程は特性方程式が共役な複素数解 a±bi をもつ場合、自己相関は周期が 2 * π * / cos[a/(a^2+b^2)^1/2] に等しい循環成分をもつ　（循環成分の数は最大p/2個）
　　　　⇒　AR(p)過程は周期を持つ循環的な自己相関構造を記述可能

　　- 定常時系列はかなりの部分が(MAなしに)ARモデルだけを用いて解析できる


- 単位根過程

　　- 原系列が非定常過程であり、差分系列が定常過程である過程

　　- 特性方程式が単位根（根１）をもつ確率過程

　　- トレンド定常過程との違い：　単位根過程のトレンドは、線形トレンドのみならず確率的トレンドをもつため、不確実性が線形的に増大　（単位根過程と線形トレンドとの差は平均的に限りなく大きくなっていく）

　　- 平均回帰的ではない　（過去の影響が消えることがない）　⇒　予測の平均二乗誤差は限りなく大きくなっていく

　　- 例：
　　　　　-　ARIMA(p,d,q)過程：　差分系列が定常かつ反転可能なARMA(p,q)過程
　　　　　- ランダムウォーク
　　　　　- 株価

　　- 単位根検定
　　　　　-　Dickey-Fuller検定
　　　　　　　- AR(1)モデルの場合の検定

　　　　　-　augmented Dickey-Fuller検定
　　　　　　　- Dickey-Fuller検定をAR(p)モデルに拡張したもの
　　　　   - 帰無仮説が「単位根あり」、対立仮説が「単位根なし」
　　　　   - 検定統計量（ｔ統計量）がｔ分布に従わないことが知られている

　　　　　-　Phillips-Perron検定
　　　　　　　- 誤差項が弱従属性（※）と分散不均一を持つ場合のノンパラメトリックな検定法　（※確率変数 u(t)とu(s)は従属しているが、|t-s|が大きくなるに従って、u(t)とu(s)がほぼ独立になる）

　　　　　-　KPSS検定
　　　　　　　- 帰無仮説が「トレンド定常過程」、対立仮説が「タイムトレンド付き単位根過程」　⇒　経済学の分析では「単位根あり」を主張したい場合が多いので、分析目的に適していることが多い



- グレンジャー因果性

　　- 時系列データにおいて、データだけから因果性の有無を判断するための概念

　　- 現在と過去の Y の値だけに基づいた将来の Y の予測と、現在と過去の Y と X の値に基づいた将来の Y の予測を比較して、
　　後者の平均二乗誤差のほうが小さくなる場合、 X(t) から Y(t) へのグレンジャー因果性が存在するといわれる

　　- グレンジャー因果性が存在しない場合、VARモデルの Y の式において X に関する係数がすべてゼロになることと同値　（F検定でグレンジャー因果性を検定できる）

　　- Y(t) から X(t) へのグレンジャー因果性が存在しないための必要十分条件は、Y(t) が X(t) の分布ラグモデルで表現できること



- 見せかけの回帰

　　- 無関係の単位根過程の間にあたかも有意な関係があり、回帰の説明力が高いように見える現象

　　- 説明変数と被説明変数が単位根過程であり、回帰の誤差項が単位根の場合、説明変数と被説明変数の関係は、見せかけの回帰になる

　　- 回避の方法
　　　  - 説明変数と被説明変数の（少なくともどちらか一方の）ラグ変数を回帰に含める
　　　  - 単位根過程に従う変数の差分をとり、定常過程にしてから解析する



- 共和分

　　- 単位根過程を用いて変数間の均衡関係を記述する方法

　　- 共和分の関係がある場合
　　  - 単位根過程の線形和が定常過程になるとき、単位根過程の間には共和分の関係がある
　　  - 説明変数と被説明変数が単位根過程であり、回帰の誤差項が定常過程の場合、説明変数と被説明変数の関係は、共和分の関係となる

　　- 単位根過程は平均回帰的でなく、長期的な予測の平均二乗誤差が限りなく大きくなっていくため、長期的な挙動を予測することは困難。
　　しかし、単位根過程 y(t) と x(t) の間に共和分関係があれば、 z(t) = y(t) - a * x(t) が定常過程となるような a が存在することになり、 
　　　　z(t) は平均回帰的で長期的な挙動は一定の精度で予測可能。

　　- グレンジャー表現定理：　共和分を含んだVAR(p)モデルは、ベクトル誤差修正モデル（VECM: Vector Error Correction Model)、つまり共和分システムの定常過程表現ができる
　　　　※ベクトル誤差修正モデルは、差分系列のVARモデルに誤差修正項が加えられたモデル
　　変数間に一方向の因果のみを仮定するのが妥当でない場合に有効

　　- 例：
　　　　　- ケインズ型の消費関数：　所得(GDP)および消費
　　　　　- 購買力平価仮説(PPP: Purchasing Power Parity)：　log(日本の物価水準) = log(アメリカの物価水準) + log(円ドル為替レート)
　　　　　- リスクプレミアム：　社債の名目金利 = 国債の名目金利 + リスクプレミアム　　※フィッシャー効果が正しいとすると　国債の名目金利 = インフレ率 + 誤差項
　　　　　- 単位根過程とその合理的期待を反映した次時点の予測値

　　- 共和分検定　（共和分関係が成り立っているかどうかの検定）
　　　　　-　Eagle-Granger検定 (共和分関係が１つであるが未知の場合）
　　　　　　　- y(t), x(t)ともに一階の差分系列が定常過程となる単位根過程であるとし、y(t)をx(t)に回帰した残差u(t)に対して単位根検定を実施
　　　　　　　　　⇒　u(t)が単位根をもつという帰無仮説（見せかけの回帰である）が棄却された場合は、y(t)とx(t)の間に共和分関係が成り立つ
　　　　　　　- 帰無仮説が「共和分関係でなく見せかけの回帰である」であり、通常の単位根検定とは棄却点が異なる

　　　　　-　Phillips-Ouliaris検定
　　　　　　　- 帰無仮説は「共和分関係なし」、対立仮説は「一次独立な共和分関係が１個ある」

　　- Johansen の共和分検定およびモデル特定化の方法：　システムが多くともh個の共和分関係しか含まないという帰無仮説を検定、ベクトル誤差修正モデルに基づいた最尤法で複数の共和分関係を同時に推定
　　　　　-　Johansen のトレース検定：　対立仮説は「すべての変数が定常」
　　　　　-　Johansen の最大固有値検定：　対立仮説は「h+1個の共和分関係が存在する」
　　　　　- 検定を繰り返し行うため検定のサイズ調整が難しいが、現実的には2つの検定をすべてのhで実施し、総合判断する



- トービットモデル

　　- 潜在変数が一定水準を上回るときにはそのまま観察されるが、一定水準以下のときにはゼロなどの特定の数値に変換されて観察されるような変数を、被説明変数にする場合に適用

　　- 例：
　　　　　- サッカー選手の試合の出場時間：　潜在変数は潜在的な能力
　　　　　- 週あたりボランティア活動時間：　潜在変数はボランティアの活動希望時間
　　　　　- 心身症状指標：　潜在変数は不健康の度合い


- ヘーキットモデル　（サンプルセレクションモデル）

　　- 潜在変数がある条件を満たすときにはそのまま観察されるが、条件を満たさないときには何も観察されないような変数を、被説明変数にする場合に適用

　　- 例：
　　　　　- 高年齢者の賃金：　就業しているという条件を満たして初めて観察される




- クロススペクトル　あるいは　クロススペクトル密度関数

　　- 相互共分散関数のフーリエ変換

　　- 帯域フィルターを通した角周波数(w)の２信号の乗積平均



- コヒーレンシー（コヒーレンス）

　　- ２信号のフーリエ周波数成分の相互相関係数の二乗

　　- ２信号間の各周波数成分ごとの線形性の程度を表す
